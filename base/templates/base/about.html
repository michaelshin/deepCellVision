About this project:

This web tool was designed to accept a raw microscopy image as input and returns the predicted protein localization for the whole image as well as a segmentation mask for the cells in the image and an estimate of protein localization for each cell. It lets users to select segmented cells or localization categories to view more detailed attributes specific to the selection. This tool is unique as there currently are other online tools that use modern computer vision algorithms to classify and segment microscopy screens.
